# Twitter-Sentiment-Analysis-Project

## Project Overview

This project analyzes Twitter data and classifies tweets as Positive, Negative, or Neutral using NLP and Machine Learning.

## Objective

To analyze Twitter data and identify whether a tweet is Positive, Negative, or Neutral using NLP and Machine Learning techniques.

## Dataset

Twitter Sentiment Dataset

## Tools Used

* Python
* Pandas
* Scikit-Learn
* Matplotlib

## Model Used

* Naive Bayes

## Results

Successfully classified tweets into Positive, Negative, and Neutral sentiments. Sentiment distribution was visualized using charts.

## Conclusion

This project helps understand public opinions and sentiment trends from Twitter data.
